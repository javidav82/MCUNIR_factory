# MCUNIR Factory Management System

A comprehensive factory management system for handling print jobs, document validation, and factory status monitoring.

## Tech Stack

- **Backend**: Laravel 8.x
- **Frontend**: 
  - Bootstrap 5
  - jQuery
  - Chart.js
- **Database**: MySQL

## Testing

Run the test suite:
```bash
php artisan test
```
